将HTML代码分解为DOM节点层次图:

HTML文档可以说由节点构成的集合，三种常见的DOM节点:
1. 元素节点：上图中html、body、p等都是元素节点，即标签。
2. 文本节点:向用户展示的内容，如li...li中的JavaScript、DOM、CSS等文本。
3. 属性节点:元素属性，如a标签的链接属性href="http://www.imooc.com"。
